## Featured project

### [CVAF-SwinB — Breast Cancer Mammography Classification](https://github.com/nirjanashrestha4/CVAF-SwinB-Mammography-Classification)

A dual-view fusion model built on Swin Transformer for binary breast cancer classification using four-view screening mammography (L-CC, L-MLO, R-CC, R-MLO).

- Designed CVAF-SwinB combining Asymmetric Lateral Fusion and Bilateral Contralateral Fusion modules
- Achieved test AUC **0.7518** on CBIS-DDSM + VinDr-Mammo (5,093 patients, ~9:1 class imbalance)
- Full ablation study across view combinations and four class-balancing strategies
- Outperforms MV-Swin-T baseline (0.7138 AUC) on the same dataset
